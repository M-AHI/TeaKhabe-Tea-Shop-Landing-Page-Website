## 2. Project Overview

The **TeaKhabe** project is a **static tea shop landing page website** designed to promote tea products and a tea-based business.
The website focuses on a **clean layout, attractive visuals, and modern UI design**.

This project is mainly created to practice:

* Frontend web design
* Utility-first CSS using Tailwind
* Layout building
* Section-based webpage design

---

## 3. Problem Description

### What problem does this project solve?

Many small businesses like tea shops need:

* A simple online presence
* A visually attractive landing page
* A way to showcase products and customer trust

This project solves the problem by providing a **ready-to-use promotional website** that highlights:

* Tea products
* Customer reviews
* Business branding

---

### Why is this project important in real life?

* Tea businesses are common and competitive
* A good website helps attract customers
* Landing pages increase customer engagement

This type of website can be used for:

* Tea shops
* Cafés
* Beverage brands
* Small food businesses

## 4. Project Objectives

The main objectives of this project are:
